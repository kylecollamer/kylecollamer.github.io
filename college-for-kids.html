College for Kids

About the Project

College for Kids (CFK) is a six-week summer enrichment program hosted each year at Penn State Behrend in Erie, Pennsylvania. It provides students with opportunities to learn, grow, and build critical skills during the summer break.

For my Penn State SWENG (Software Engineering) 480 and 481 classes, this group capstone project was to develop a digital system for all CFK items.
